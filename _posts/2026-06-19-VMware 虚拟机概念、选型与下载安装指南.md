## 什么是虚拟机？

在了解 VMware 之前，我们先理解一个概念：**虚拟机（Virtual Machine，简称 VM）**。

简单来说，虚拟机就是**用软件模拟出来的一台完整电脑**。它拥有自己的虚拟 CPU、内存、硬盘和网卡，可以独立安装和运行操作系统（如 Windows、Linux、macOS），就像一台真实物理机一样。

但与真实电脑不同的是，虚拟机是**运行在你当前电脑上的一个应用程序**，并且与宿主机系统相互隔离。这意味着你可以在虚拟机里"为所欲为"——安装各种软件、测试危险代码、折腾系统配置——而不用担心搞坏自己的主力电脑。

## VMware 能做什么？

以下是 VMware 虚拟机最常见的几个应用场景：

- 🖥️ **跨系统办公**：在 Windows 上使用 Linux 开发环境，或在 Mac 上运行仅支持 Windows 的专业软件。
- 🔒 **安全隔离测试**：可疑文件、钓鱼软件、未知补丁，统统先在虚拟机里跑一遍，安全无忧。
- 📸 **系统快照与回滚**：像游戏存档一样，在关键操作前拍下"快照"，出错了随时一键恢复。
- 🧪 **搭建实验环境**：模拟多台服务器、集群、路由器，用于学习网络架构或分布式系统。
- 📦 **软件兼容性测试**：在同一台电脑上同时运行多个不同版本的操作系统，测试软件兼容性。

## 安装与初次使用指引

### 1. 安装步骤

- 下载对应平台的安装包（`.exe` / `.dmg` / `.bundle`）；
- 双击运行，按安装向导提示操作；
- 大多数情况下，一路"下一步"即可完成安装；
- 建议将安装路径自定义到非系统盘（如 D 盘），避免占用 C 盘空间。

### 2. 创建第一台虚拟机

安装完成后，打开软件，点击 **"创建新的虚拟机"**，按以下流程操作：

1. 选择安装来源（光盘镜像文件 `.iso` 或物理光驱）；
2. 选择要安装的操作系统类型（Windows / Linux / 其他）；
3. 分配虚拟硬件资源（CPU 核心数、内存大小、磁盘容量）；
4. 启动虚拟机，进入操作系统安装流程（与真实电脑安装系统完全一致）。

> 📌 **新手建议**：内存分配不要超过宿主机实际内存的一半，磁盘建议选用"动态扩展"模式，节省实际硬盘空间。
